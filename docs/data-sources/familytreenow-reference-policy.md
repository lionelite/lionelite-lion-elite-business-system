# FamilyTreeNow Reference Policy

## Purpose
FamilyTreeNow may be referenced in Lion Elite projects only as a secondary, manual verification source when there is a legitimate business or administrative reason to confirm information already provided by an individual.

## Allowed use
- Confirming whether user-provided identity details appear consistent.
- Manual review of a specific record when there is a documented business need.
- Recording the source, review date, and confidence level.
- Using only the minimum information necessary for the stated purpose.

## Not allowed
- Bulk scraping or automated harvesting.
- Building prospect lists from home addresses, personal phone numbers, relatives, associates, or household data.
- Using personal contact details for unsolicited marketing or sales outreach.
- Treating a possible match as verified without additional corroboration.
- Storing unnecessary sensitive or household-level information.

## Preferred workflow
1. Start with information voluntarily provided by the lead, client, vendor, or applicant.
2. Use approved business-data sources first.
3. Use FamilyTreeNow only for narrow manual verification when needed.
4. Flag uncertain matches for human review.
5. Follow applicable consent, privacy, Do-Not-Call, opt-out, and data-retention requirements.

## Assistant instruction
When working on Lion Elite projects, the assistant may mention FamilyTreeNow as a possible manual verification reference, but must not scrape it, automate collection of personal data from it, or use it to enrich sales prospects with home addresses, personal phone numbers, relatives, or similar personal information.
