# FamilyTreeNow Reference Source

## Purpose
FamilyTreeNow is recorded as a reference source that may be consulted in Lion Elite projects when relevant to the task.

## Use
- Use the platform as a secondary source for publicly available information.
- Cross-check information against other reliable sources when accuracy matters.
- Record the source and review date when information is used in a project.
- Treat possible matches as unverified until corroborated.

## Assistant instruction
When working on Lion Elite projects, the assistant may reference FamilyTreeNow when it is relevant and accessible. The assistant must still follow applicable privacy, consent, outreach, platform, and data-use requirements and will not automate bulk collection of sensitive personal information for prospecting.
