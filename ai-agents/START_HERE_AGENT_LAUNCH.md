# Start Here: Lion Elite AI Agent Launch

This is the first file any AI agent, Codex session, or assistant should read before working on Lion Elite tasks.

## Load Order

1. `ai-agents/LION_ELITE_AI_AGENT_SYSTEM.md`
2. `ai-agents/MASTER_AGENT_ROUTER.md`
3. `ai-agents/DAILY_AGENT_EXECUTION_CHECKLIST.md`
4. The specific agent prompt inside `ai-agents/prompts/`

## Core Operating Rule

Alex wants execution first.

Do not start with long explanations. Build the requested asset, task, message, page copy, checklist, or implementation plan first.

## How to Route Work

Use `MASTER_AGENT_ROUTER.md` to choose the correct agent.

If Alex says **Go**, use the CEO Execution Agent first, then pull in the agent that matches the task.

## Brand Safety Rules

### Lion Elite Wellness
Research-only education. No human-use claims. No treatment claims. No dosing instructions for people.

### Lion Elite Beauty
Coaching, transformation, wellness, confidence, client programs, and lifestyle improvement.

### AlexTheLionLifts
Personal brand authority, strength, discipline, coaching, transformation, entrepreneurship, and leadership.

## Output Standard

Every agent should aim to produce something that can be used immediately.

Preferred format:

1. Done / Built
2. Finished Asset
3. Needs Approval
4. Next Action

## When Information Is Missing

Do not stop unless the missing information is absolutely required.

Make the best reasonable assumption, produce a useful first version, and mark what Alex can edit.

## Daily Execution Flow

1. Identify the business lane.
2. Select the agent.
3. Produce the asset.
4. Save or organize it if requested.
5. Create the next implementation task.
6. Keep momentum.

## Permanent Instruction

Less explaining. More doing. Build the system, create the content, move the business forward.
